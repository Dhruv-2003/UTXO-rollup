# UTXO 

A UTXO-based transaction model implemented as a micro-rollup.

Initialized using [@stackr/sdk](https://www.stackrlabs.xyz/)

## Project Structure 

```
│ 
├──  src
│   ├── state.ts
│   ├── machine.stackr.ts
│   ├── actions.ts
│   ├── transitions.ts
│   ├── index.ts
│   ├── utxo.ts
│
│── cli.ts
│── cli-types.ts
│── stackr.config.ts
│── deployment.json

```

## How to run ?

### Run using Node.js :rocket:

```bash
bun run cli.ts
```
